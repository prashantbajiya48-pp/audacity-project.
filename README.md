# Audio Editing and Noise Reduction using Audacity

## Project Overview
This project demonstrates how audio files can be edited and improved using Audacity. 
Audacity is a free and open-source audio editing software that allows users to record, edit, and enhance sound recordings.

The project explains the process of audio editing, noise reduction, and applying audio effects.

## Objectives
- To understand the basics of audio editing
- To learn how to use Audacity software
- To perform noise reduction on audio recordings
- To demonstrate audio editing techniques

## Software Used
Audacity (Open Source Audio Editing Software)

## Features
- Audio Recording
- Audio Cutting and Trimming
- Noise Reduction
- Applying Sound Effects
- Exporting audio in multiple formats

## Project Files
- Presentation.pptx – Project presentation slides
- Project_Report.docx – Detailed project report
- Screenshots – Images of audio editing process
- SampleAudio.mp3 – Example audio file

## Steps Performed
1. Installed Audacity software
2. Recorded or imported an audio file
3. Edited the audio using cutting and trimming tools
4. Applied noise reduction to remove background noise
5. Exported the final edited audio

## Advantages of Audacity
- Free and open-source
- Easy to use interface
- Supports multiple audio formats
- Suitable for beginners and professionals

## Conclusion
Audacity is a powerful and user-friendly audio editing software. 
This project demonstrates how audio recordings can be improved using different editing tools and noise reduction techniques.
